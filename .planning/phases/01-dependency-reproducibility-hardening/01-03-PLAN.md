Recovery facts (from RESEARCH.md Recovery Path Runbook):
- `uv sync --frozen` restores exactly what uv.lock specifies (refuses to update).
- uv's local cache `~/.cache/uv/git-v0/` may retain a git repo even after upstream deletion.
- Per-source state after Phase 1: openhomedevice = user-owned fork on immutable SHA (no
  mirror needed); pyatv = PyPI 0.17.0 immutable wheel (no mirror needed); all other deps =
  exact versions in uv.lock.
- Future migration trigger: when `bazwilliams/openhomedevice` publishes a release >=2.4.0
  on PyPI WITHOUT lxml, switch openhomedevice to `>=2.4.0,<3` and drop the fork.
